# TechKart 🛒

TechKart is a modern **React-based eCommerce frontend application** designed to simulate a real online tech store.  
It provides an intuitive interface where users can browse products, search items, view product details, and manage their shopping cart.

The project focuses on **clean UI design, responsive layouts, and smooth navigation using React Router.**

---

## 🚀 Features

- 🏠 Product browsing interface
- 🔍 Real-time search functionality
- 📦 Product details page
- 🛒 Shopping cart management
- 🔢 Cart item counter in navbar
- 🔐 Login & Register UI pages
- 📱 Fully responsive design
- ⚡ Fast page navigation with React Router
